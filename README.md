# Employee Management System

A modern Employee Management System built using React, Vite, Tailwind CSS, Context API, and Local Storage. The application allows administrators to assign tasks to employees while enabling employees to manage and track their assigned work.

## Features

- Admin Authentication
- Employee Authentication
- Task Assignment
- Task Status Tracking
- Dashboard with Task Statistics
- Responsive User Interface
- Local Storage based Data Management

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- Context API
- JavaScript (ES6+)
- Local Storage

## Installation

```bash
git clone https://github.com/keshavpatel642/employee-management-system.git

cd employee-management-system

npm install

npm run dev
```
